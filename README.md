## Projeto Jogo de Cartas - Gerra de Pontos

Conceito do Jogo: "Guerra de Pontos"
Objetivo: Acumular mais pontos que o oponente ao final de um número definido de rodadas.

Preparação: Um baralho é criado e embaralhado.

Rodadas:

A cada rodada, um jogador recebe uma carta.

As pontuações das cartas são comparadas.

O jogador com a maior pontuação na carta vence a rodada e ganha 1 ponto. Em caso de empate, ninguém pontua.

Fim de Jogo: O jogo termina após um número pré-definido de rodadas (ex: 5 rodadas).

Vencedor: O jogador com a maior pontuação total ao final das rodadas é o vencedor.
